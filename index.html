Bayesian Conditional Cointegration

Due to popular demand, this website has been created to share code for the paper Bayesian Conditional Cointegration presented at ICML 2012.

So far, I have created an iPython notebook which you can view with nbviewer or even run with mybinder

If there is interest, I will try to find the time to release versions of my code in MATLAB, R, or even C. If you are interested, please get in touch.

I hope this code proves useful.
